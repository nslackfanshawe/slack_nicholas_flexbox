# slack_nicholas_flexbox
This is the submission for the nested grid, flexbox, assignment for web development 1. 

## Description
This project is a responsive hero section using flexbox and nested grid. This is based off of the Dr. Nut brand that will be used for the FIP.

## Features
- Responsive hero section layout.  
- CSS Grid for overall structure.  
- Flexbox for navigation.
- Responsive images using `srcset`.  
- Mobile-first design.
- Responsive nav.

## Technologies Used
- HTML  
- CSS  
- JS
